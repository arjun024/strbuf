strbuf
======

standalone version of git's strbuf api

####API####
Please refer to git's strbuf api:

https://www.kernel.org/pub/software/scm/git/docs/technical/api-strbuf.html

####Compiling####
```
$ gcc test.c strbuf.c -o executable
```
